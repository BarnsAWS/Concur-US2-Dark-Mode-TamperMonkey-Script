# About

Tampermonkey userscript that re-themes `us2.concursolutions.com` (SAP Concur expense / travel) using the Cloudscape v3.3 dark-mode token palette. Standard seven-phase engine — see `STANDARDS.md`. Preserves SAP Concur brand blue logo, sign-in hero photo (media hands-off), and brand-color CTAs. Auth-gated; verify changes after sign-in.

Per the dual-update rule in `STANDARDS.md`, this repo's changes also land in the bundle.
